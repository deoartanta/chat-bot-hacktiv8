# 🍳 Chatbot Kuliner AI - Resep Nusantara

Chatbot berbasis AI yang membantu pengguna menemukan resep masakan Indonesia, memberikan panduan memasak, serta tips kuliner secara interaktif seperti berbicara dengan chef.

---

## 🚀 Fitur Utama

- 🥘 Rekomendasi resep masakan Indonesia
- 📋 Langkah memasak yang jelas dan terstruktur
- 🔪 Tips teknik memasak
- 🌶️ Takaran bahan & bumbu
- 🔄 Alternatif bahan pengganti
- 💬 Interaksi chatbot sederhana dan responsif

---

## 🧰 Teknologi yang Digunakan

- Node.js
- Express.js
- HTML, CSS, JavaScript (Vanilla)
- JSON (data resep)
